🌍 PostGIS Magic

The geospatial superpower. One SQL query finds nearest shops:

SELECT * FROM shops
ORDER BY location <->
ST_Point(lng, lat)
LIMIT 10;

This is why we chose Supabase over Firebase — Firebase can't do this natively.

📦 Lucknow Demo Data Plan

Phase 1 — Import (Day 1)

Pull Lucknow shop data from OpenStreetMap Overpass API. Auto-import into Supabase. Gets us ~200-400 shops instantly, mostly in main areas.

Phase 2 — Curate (Days 2-3)

You manually verify & enrich 3 Lucknow areas: Hazratganj, Gomti Nagar, Indira Nagar. Add phone numbers, hours, audio scripts for key shops.
