Bhaiya App — Design Section 1: Core Concept & Screens

The full picture of what we're building for the Lucknow demo

🎯 The Core Promise

A hyperlocal discovery app that works like a trusted local bhaiya — finds small shops, gives you their number, tells you if they're open, and guides you there. Built for people starting fresh in a new city.

📱 The 7 Screens

1️⃣

Splash / Onboarding

"Naye sheher mein? Bhaiya hai na!" — sets the emotional tone. City selector. Language toggle (Hindi/English).

2️⃣

Home

GPS-detected location. Search bar. Category grid (ration, sabzi, milk, hardware…). Nearby open shops strip.

3️⃣

Map View

Leaflet + OpenStreetMap. Colour-coded pins by category. Tap pin → mini shop card. "Nearest open" button.

4️⃣

Shop List

Category filtered. Sorted by distance. Open/closed badge. Distance in metres. Rating stars.

5️⃣

Shop Detail

Name, address, phone (tap to call), hours, open now badge. 🔊 Bhaiya Audio Guide button. Directions on map.

6️⃣

Add a Shop

Community contribution form. Drop pin on map. Fill details. Submitted for admin review. "Help your neighbours!"

7️⃣

Admin Panel

Review submitted shops. Approve / reject. Edit details. Bulk import from OpenStreetMap. Data dashboard.

🔊 The Bhaiya Audio Feature

Uses the browser's built-in Web Speech API (free, no API cost). On the shop detail page, tapping "Bhaiya se puchho" plays a voice guide in Hindi:

"Seedha jaao 300 metre, phir left lena. Wahan ek red building hai — uske paas Sharma General Store milega. Subah 8 baje se raat 9 baje tak khula rehta hai."

For demo: pre-scripted for 10 key shops in Lucknow. Post-funding: AI-generated for all shops.
